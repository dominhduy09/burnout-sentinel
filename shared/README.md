# Shared

This folder is reserved for code or assets shared across the frontend, backend, and future ML pipeline.

Use this folder when you need one source of truth for values that should stay aligned across the app.

Good future uses:

- shared feature names
- prompt templates for recommendation generation
- example payloads
- constants for healthy workload thresholds

Possible additions later:

- shared TypeScript/Python schema references
- API payload examples used in docs and tests
- threshold constants for workload, sleep, and recovery states
- reusable prompt text for generated recommendations or summaries
